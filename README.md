# 📊 Profit Intelligence Dashboard

## 🚀 Project Overview

The **Profit Intelligence Dashboard** is an end-to-end business analytics project focused on uncovering hidden profitability drivers in supply chain and sales operations. Instead of analyzing only revenue, this project evaluates:

- Customer profitability
- Product & category margins
- Discount-driven margin erosion
- Market and regional profitability
- Revenue vs profit behavior

The dashboard enables interactive business insights using **Python, Streamlit, Pandas, and Plotly**.

---

# 🎯 Business Problem

Organizations often achieve strong revenue growth but struggle with weak profitability due to:

- Excessive discounting
- Low-margin products
- Unprofitable customers
- Regional inefficiencies

This project helps identify:

✅ Which customers generate real profit  
✅ Which products leak margin  
✅ Which markets are revenue-heavy but profit-weak  
✅ At what discount threshold margins decline
